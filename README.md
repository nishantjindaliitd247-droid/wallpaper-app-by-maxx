# 🖼️ Wallpaper App

A modern wallpaper gallery built with React, Vite, and Axios. The app fetches wallpapers from an API and displays them in a clean, responsive interface with pagination.

## 🚀 Live Demo

https://gallery-projectbymax.netlify.app

## ✨ Features

- Browse wallpapers
- API Integration
- Pagination
- Responsive Design
- Fast performance with Vite

## 🛠️ Tech Stack

- React
- Vite
- Axios
- CSS

## 📦 Installation

```bash
git clone <repository-url>
cd wallpaper-app
npm install
npm run dev
```
